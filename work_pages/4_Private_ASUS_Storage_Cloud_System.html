資源分享平台 by Private ASUS Cloud

台南一中教師資源分享平台主要係透過 ASUS Cloud API 存取 ASUS Cloud中的資源，分享給家長、學生察看與下載。而ASUS Cloud API主要是依循Restful API進行設計，由於private ASUS Cloud 並沒有提供任何SDK，因此在開發過程中需要根據開發文件的格式逐一撰寫對應的處理，是一次全新開發經驗。
