大一 程式設計作業 - 使用C/Java實作簡易資料庫軟體

大一程式設計課，授予 C 語言和 Java 兩大程式語言，並以該程式語言開發 了心理學者相關資料的管理軟體.
C : 語言部分，主要是利用迴圈判斷式等方式處理相關的操作邏輯，並利用檔案存取以及自訂的資料結構來協助儲存與顯示處理相關資料.
Java : 部分透過Java swing 操作視窗，更加有應用程式的使用體驗，並且初步了解程式的按鈕觸發機制.
